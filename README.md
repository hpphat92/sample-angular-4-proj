## Install Notes
1. open package.json file

2. Under dependencies, remove these two values: <br>
  "ammap3": "github:amcharts/ammap3",<br>
  "amcharts3": "github:amcharts/amcharts3",<br>

3. Then run npm install. It will work just fine and create a folder by the name of node_modules<br>
  Inside node_modules folder, clone the following repositories:<br>
  git clone https://github.com/amcharts/ammap3.git<br>
  git clone https://github.com/amcharts/amcharts3.git<br>
